# Group-div

This project is inspired from instagram and allow anyone to login with google and post images and comments

[![Deployed group-div](https://vercel.com/button)](https://group-div.vercel.app/)


-----------------------
----Technology used----
-----------------------
- Next.js(React)
- Tailwind CSS
- Firebase (ver-9)
- Recoil

Data is been generated by Faker.js </br>
Icons are by HeroIcons </br>
NextAuth.js for sign-in authentication</br>
headlessui is used </br>
react-moment for timestamps </br>
